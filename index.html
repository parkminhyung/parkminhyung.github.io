Multi-Market Alpha Lab

Systematic multi-factor equity research across KRX, JPX, and CHN.

Multi-Market Alpha Lab은 KRX·JPX·CHN 시장을 대상으로 한 체계적 멀티팩터 퀀트 리서치 공간입니다.

모델의 구조와 팩터 설계는 GitHub 레포지토리 Multi-Factor-Alpha-Strategy에 공개되어 있으며, 모멘텀·밸류·퀄리티 등 복수의 팩터를 결합한 앙상블 구조로 설계되어 있습니다. 매매 시그널은 매일 자동으로 생성되며, 포지션 진입·청산·리밸런싱이 전략 규칙에 따라 실행됩니다.

본 사이트의 Performance 페이지에 표시되는 가상매매 성과 지표(수익률·MDD·Sharpe 등)와 Downloads 페이지에서 내려받을 수 있는 일별 거래 로그·NAV CSV는, 모두 위 GitHub 레포지토리에 공개된 모델을 그대로 실제로 운용한 라이브 테스트의 결과입니다.

 

Multi-Market Alpha Lab is a systematic multi-factor quantitative research platform covering KRX, JPX, and CHN equity markets.

The model architecture and factor design are publicly available on GitHub: Multi-Factor-Alpha-Strategy. The strategy combines multiple factors — including momentum, value, and quality — through an ensemble framework to generate systematic alpha signals. Trading signals are generated automatically on a daily basis, with position entries, exits, and rebalancing executed according to predefined strategy rules.

All performance metrics shown on the Performance page (returns, MDD, Sharpe ratio, etc.) and the daily trade logs / NAV data downloadable from the Downloads page are produced by running the exact model published in the repository above — they are live-test results, not backtests.

 

Multi-Market Alpha Lab 是一个面向 KRX（韩国）、JPX（日本）及 CHN（中国）股票市场的系统性多因子量化研究平台。

模型的架构与因子设计已在 GitHub 仓库公开：Multi-Factor-Alpha-Strategy。该策略通过集成框架融合动量、价值、质量等多个因子，系统性地生成阿尔法信号。交易信号每日自动生成，仓位的建立、平仓及再平衡均依据预设策略规则执行。

Performance 页面所展示的业绩指标（收益率、最大回撤、夏普比率等），以及 Downloads 页面提供下载的每日交易记录与净值数据，均为上方 GitHub 仓库公开模型的实盘运行（live-test）结果，并非回测结果。
